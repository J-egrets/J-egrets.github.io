0 电商系统

为了巩固自己JavaWeb阶段的知识，我选择了一个较为简单的电商项目来练手。

1 项目概述

本项目从电子商务最基本的需求出发，将系统分为前台系统和后台系统，前台系统是消费者使用的子系统，后台系统是管理者使用的子系统，以下是两个子系统的各个功能展示。

2 数据库设计

2.1 表详细设计

表t_user，用户表

字段名 字段类型 字段中文名 备注
userName varchar(255) 用户名 主键
password varchar(255) 密码 无
gender varchar(255) 性别 无
phone varchar(255) 电话 无
address varchar(255) 地址 无
status int(11) 是否冻结 0表示正常 1表示冻结
role int(11) 用户身份 0表示管理员 1表示普通用户

表t_goods_categories，商品类别表

字段名 字段类型 字段中文名 备注
id varchar(255) 类别id 主键
name varchar(255) 类别名称 无

表t_goods，商品表

字段名 字段类型 字段中文名 备注
id varchar(255) 商品id 主键
name varchar(255) 商品名字 无
description varchar(255) 商品描述 无
price 商品价格 无
img varchar(255) 商品图片 无
count int(11) 商品库存 无
categoryId int(11) 所属类别编号 无

表t_car，购物车表

字段名 字段类型 字段中文名 备注
id int(11) 购物车id 主键
userName varchar(255) 购物车用户名 无
goodsId varchar(255) 货品id 无
goodsName varchar(255) 货品名 无
goodsPrice decimal(10) 货品单价 无
goodsImg varchar(255) 货品图片 无
number int(11) 货品数量 无
totalPrice decimal(10) 总价 无

表t_order，订单表

字段名 字段类型 字段中文名 备注
id varchar(255) 订单id 主键
userName varchar(255) 下单用户名 无
goodsId varchar(255) 商品Id 无
count int(11) 下单商品数量 无
status int(11) 订单状态 0表示审核中 1表示通过 2表示未通过
orderTime dateTime 付款方式 无
sendType varchar(255) 发货方式 无
payType varchar(255) 付款方式 无
totalPrice decimal(10) 订单总金额 无
